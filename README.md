# AI Software Factory CLI

Standalone terminal-based AI software engineering team orchestration platform.

This repository currently contains the Phase 0, Phase 1, and Phase 2 implementation under
[`ai-cli/`](/Users/shinobi/bibek_code/Omnix-CLI/ai-cli): CLI foundation,
project initialization, validated project state, the model-agnostic provider
layer, and a state-aware Master Agent with persistent conversations, goals, and
decisions.

The product is intentionally separate from Omnix internals.
